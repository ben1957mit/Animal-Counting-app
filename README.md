# Animal-Counting-app
# 🐾 Animal Counting App (Ages 2–4)

A simple, touch‑friendly counting game for toddlers using animals, stars, and positive rewards.  
Built with Streamlit so it works on tablets, phones, and computers.

## 🎮 Features
- Big animal visuals (emoji or PNG images)
- Tap‑friendly number buttons
- Randomized counting challenges
- Reward stars for correct answers
- Confetti celebration after 5 stars
- Works on iPad, iPhone, Android, Windows, Mac

## 🚀 Run Locally
Install Streamlit:

    pip install streamlit

Run the app:

    streamlit run app.py

## 🌐 Deploy on Streamlit Cloud
1. Push this repo to GitHub  
2. Go to https://share.streamlit.io  
3. Select this repo  
4. Deploy  
5. Share the link with kids, parents, or teachers

## 📁 Assets
Place optional PNG animal images in:

    assets/animals/

Place optional sound effects in:

    assets/sounds/

## 👶 Designed For
Children ages 2–4 learning to count using animals and positive reinforcement.
